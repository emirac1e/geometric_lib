# Документация проекта Geometric Library

## Общее описание
Данный проект представляет собой небольшую библиотеку для вычисления геометрических характеристик простых фигур — круга и квадрата.  
Библиотека может быть использована в учебных целях или в качестве вспомогательного модуля для более сложных вычислений.  s
Каждая функция снабжена комментариями и примером вызова.

---

## Описание функций

### Модуль `circle.py`

#### `area(r)`
Вычисляет площадь круга по заданному радиусу.  
**Формула:**  
\[
S = \pi r^2
\]

**Пример использования:**
```python
area(3)  # 28.274333882308138
```

---

#### `perimeter(r)`
Вычисляет длину окружности (периметр круга) по радиусу.  
**Формула:**  
\[
P = 2\pi r
\]

**Пример использования:**
```python
perimeter(3)  # 18.84955592153876
```

---

### ⬜ Модуль `square.py`

#### `area(a)`
Вычисляет площадь квадрата по длине стороны.  
**Формула:**  
\[
S = a^2
\]

**Пример использования:**
```python
area(3)  # 9
```

---

#### `perimeter(a)`
Вычисляет периметр квадрата по длине стороны.  
**Формула:**  
\[
P = 4a
\]

**Пример использования:**
```python
perimeter(3)  # 12
```
